This section should be useful if you want to see how models made around the same time share characteristics or inspiration. You should be able to see how my technique and style have progressed from the very beginning.

I have also provided comments for each model that may include information about its name, inspiration, or technology. Clicking the preview image for each model opens a live constructor window so you can see it in action.

201 name: Crux
genre: roller
comments: The relative closeness of the wheels in this model makes it want to flip over as it rolls. I originally had the wheels anchored to tension springs that were on the exact opposite sides of the motor, but it got hung up on the ground so I used a position that is a few degrees off to raise it up.
202 name: 4pt_Angle_Counter_Rotor
genre: motor
comments: The 4pt Rotation Angle Motor is a configuration that I have used extensively since the introduction of this model. This version uses true linear motors with phased supports but sometimes I use a compound, embedded, or anchored arrangement.
203 name: Land_Cruiser
genre: walker
comments: This model uses my 4pt_Angle_Counter_Rotor in an inertial setup to drive rotating platforms. To me it looked like pontoons or the hull of a ship so I gave it a nautical sounding name.
204 name: Juxtaposition
genre: walker/motored
comments: The design of this model juxtaposes circle shapes with sharp lines intersecting at right angles. Also, the legs on one side are offset from the other side to give it a more interesting look.
205 name: Photonic
genre: zero-g/motored
comments: This is the first time I really tried to tackle the style of model pioneered by Jeckyll. The real trick is to find the lengths necessary to make compound and embedded linear motors whose power points line up over the same distance. Other than that it just requires patience to construct.
206 name: Cyan
genre: walker/motored
comments: Cyan is a model that I was most proud of for a long time, although I have since surpassed it. To me it is the model that quintessentially defines my style. It uses a compound linear combination to produce four counter rotors links by tension springs that anchor the legs.
207 name: Prowess_JeckyllStyle
genre: roller/motored
comments: Jeckyll has always been known for his fantastic linear combinations but this model is based on his new method of saturation. In effect it leaves no motor ends hanging of the structure like in some other combinations. I simply copied his form and linked two structures with a tension spring down the middle.
208 name: Thrinax
genre: roller
comments: Making a turbine combination is not actually too different from making a linear motor combination once you understand the timing involved. This model is great because it uses flex circles like gears to increase the advantage of the turbines.
209 name: Introspect
genre: walker
comments: This model is very simple in operation and is like many of my mechanical walkers that use offset timing. I built it because I love the idea of using intricate tension spring frames that leave large open spaces where some sort of structure normally is.
210 name: Inscribed
genre: other/flex
comments: This might be my only model that doesn't move whatsoever. It might be a little difficult to appreciate until you have done a fair bit of constructing yourself, but it is very hard to match the sizes of these complex tension springs and flex shapes. There is no secret, it just took a lot of trial and error.
211 name: Turbox
genre: roller
comments: Turbox is related to Thrinax but each end of the turbines are connected to different size rings and are arranged in a square. The real problem is programming. You can't just hook up the turbines, they must all be in the proper phase. I made several mistakes before I got it right.
212 name: Facticity
genre: roller/motored
comments: Like the model, Facticity is a simple sounding philosophical concept but unlike it carries profound implications. This is a model that I had the idea for the first day I started working with the constructor. Sadly, I didn't have the skill or understanding at the time to make it so it seems like an anachronism here.
213 name: Stilt
genre: walker
comments: About this time I ran into a stagnant period. Whenever my imagination slows down I like to return to very old structures or delve into the zoo's past. This resulted in a series of models that use daintywalker style legs to drive longer tension spring legs.
214 name: Amper
genre: walker
comments: Like Stilt, this model resulted from experimenting with relatively ancient structures to get my imagination going again. I don't know where the name came from other than I thought it sounded interesting. I suppose the naming process was similar to Antler's as well.
215 name: dirkwalker
genre: walker
comments: There are many daintywalker tribute models out there but not many dirkjiggler ones. This resulted from a challenge in the forum that people make walkers based on the twelve original models. Of course, I had to construct a new complimentary dirkjiggler to work with the original one.
216 name: Combine
genre: walker
comments: Combining different motor types can sometimes produce hideous looking models. But teaming these two different rotor types actually turned out okay. The only main problem is the slight swaying it suffers as it walks.
217 name: Machination
genre: walker
comments: Machination is a philosophical concept put forward by Martin Heidegger in his Beitraege. This model does its best to portray it by matching four different leg designs to walk in unison. The result looks a little like Frankenstein's monster, but maybe that's why the concept model idea works so well.
218 name: Fusion_of_Horizons
genre: walker
comments: This idea came from a philosopher named Hans-Georg Gadamer. I felt the middle of the structure looked like the wavy reflection that water produces. It is rare that I would use such a large amount of superfluous structure in a design but it works in this case.
219 name: Praxis
genre: walker
comments: Praxis is a Greek word and concept. The model is actually quite primitive in operation. However, the legs are made from linear motors so they appear to be shifting their appearance as they move. I consider this model to be highly visually effective.
220 name: Idiom
genre: walker
comments: Idiom is another simple walker related to Praxis. The neat part is that the three points used to connect all the leg are not actually connected to each other. This motor structure in the middle is the very first counter rotor stack, although its functionality is not used here.
