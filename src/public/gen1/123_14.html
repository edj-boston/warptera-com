This section should be useful if you want to see how models made around the same time share characteristics or inspiration. You should be able to see how my technique and style have progressed from the very beginning.

I have also provided comments for each model that may include information about its name, inspiration, or technology. Clicking the preview image for each model opens a live constructor window so you can see it in action.

261 name: Deciduous
genre: walker/motored
comments: This model looks like a pinecone to me. It is similar to the gigantic RAM I used in Helios and other models except instead of tension springs it uses huge triangles to produce elliptical motion.
262 name: Quadriceps
genre: walker/motored
comments: I didn't even realize it until I was finished and it was time to pick a name that this model only uses four muscles. It has a single 4pt linear compound linear motor and the three linkages and connecting rods do the rest of the work.
263 name: Sisyphus
genre: roller/inertial
comments: This model is named for the great tragic figure of Greek myth. Sisyphus only had to push one boulder up a hill for eternity but this motor has two boulders to contend with. I like the design of the wheels a lot and this was the first time I used them.
264 name: Newtonian_Time
genre: other/motored
comments: This model begins not only my best series of models, but also perhaps my best models period. The counter rotor stack multiples the wave pallet so the minute hand can run at twelve times the rate of the hour hand.
265 name: Einsteinian_Time
genre: other
comments: I had multiple iterations of this model going at once. Relativity is a difficult concept to present visually so I decided that showing the curvature of space-time would be the most effective. Ironically, this is most foreign to our everyday concept of time.
266 name: Phenomenological_Time
genre: other/physics
comments: This model is very subtle. It is meant to make the viewer understand how human our experience of the passage of time really is. This concept stands is stark contrast with the first two intellectual understandings of time that are just abstractions of our true experience.
267 name: Into_the_Looking_Glass
genre: other/representational
comments: It is impossible for me to follow models like my time series with anything that won't get over shadowed. As a result I decided to do something fun and as an experiment, I tried to replicate the appearance of the constructor interface using fixed and free masses.
268 name: Indigo
genre: walker/motored
comments: This model is essentially the same model as High_Tower except the motor braces themselves are programmed. It is a process called skewing that shifts the motors to a degree dependant on the relative amplitude of the brace programming.
269 name: Totum
genre: walker/motored
comments: This model uses linear motor combinations, a compound MultiMotor, linear motor bracing, and tension spring legs to achieve forward progress. For some reason I was thinking of a totem pole or other American Indian artifacts when I was making this.
270 name: Splay
genre: walker/motored
comments: To splay means to spread and that is appropriate for this model whose principle functionality is close to Dachshund's. The real trick was to find the right relative amplitude, and hence the right bracing angle, to keep the motors perfectly parallel and undistorted.
271 name: Tangent
genre: walker
comments: This model uses skewed motor bracing just like most of the models from this period. I have no idea why I named it Tangent, I guess I just always liked that word. Strangely, this is one of the few times I've used anchored linear motors.
272 name: Inertia
genre: zero-g/physics
comments: I like this model quite a bit because it demonstrates how durable and quick RAM linkages can be. Of course, you must use the highest level of craftsmanship because any inaccurate lengths can throw a monkey wrench into the whole thing.
273 name: Overkill
genre: walker/motored
comments: I'm not sure if people really understood this model. Basically it converts the linear motion from MultiMotors into rotary motion via the RAM linkages, then converts it back to linear motion by removing the vertical component with actuators. Then the legs are hooked up to regain elliptical motion.
274 name: Cotangent
genre: walker
comments: Cotangent is a relative of Tangent. I don't like the motor as much but the leg design makes it a much better model overall. And just like Splay, it required finding the right relative amplitude to avoid distorting the motors.
275 name: Method
genre: walker/motored
comments: The name Method is rather plain but I thought of it while reading Hans-Georg Gadamer's "Truth & Method". It has 3pt RAM linkages and tension springs legs. But I don't particularly like the swaying motion because it results from each set of legs being out of phase with each other.
276 name: Palindrome
genre: walker/motored
comments: While this model looks like two simple linked walkers, try picking it up or decreasing gravity. It is the program timing that keeps them anchored to each other in such a rigid fashion.
277 name: Centric
genre: other/physics
comments: Lectvay pioneered zero friction models, but I try my hand at them every once in a while if I can think of a good enough idea. This model used two concentric flex circles as mass placement guides that were deleted prior to submission.
278 name: Strobe
genre: other/physics
comments: This model used the same flex ring template as Centric. I don't think it produces as nice an effect but it certainly is hypnotizing if you stare into long enough.
279 name: Omnibus
genre: walker/motored
comments: One thing I found while practicing skewed motor braces is that they keep such perfect mass placement that sometimes you don't even need the other motor. The only disappointing thing about this otherwise fine walker is that the same thing could be achieved much easier with simple triangles.
280 name: Svensk
genre: walker/flex
comments: I always really like lonelyswedish's winning model in my guideline contest. I decided to continue the idea without the limitations of the contest and came up with this model. Incidentally, Sevens is Swedish for Swedish.
