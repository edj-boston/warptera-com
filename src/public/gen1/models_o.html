This section should be useful if you are looking for a particular model. If you'd like to see my models in the order they were made click the '123' tab above.

Besides the name, you'll see some genre information that is just meant to be a quick guide, not an absolute categorization. The comments provide a short description or discussion of the model's inspiration. Clicking the preview image will open a constructor window.

330 name: Olde
genre: zero-g/motored
comments: Olde is very similar to Spokes but each wheel contains two sets of points that are 60 degrees off from each other. For this reason, three identical wheels interlock and spin around each other. It also rotates inertially but it is hard to tell how much because of how busy the mechanism is.
279 name: Omnibus
genre: walker/motored
comments: One thing I found while practicing skewed motor braces is that they keep such perfect mass placement that sometimes you don't even need the other motor. The only disappointing thing about this otherwise fine walker is that the same thing could be achieved much easier with simple triangles.
246 name: One_Cylinder
genre: roller/motored
comments: Just for fun one day I showed a 1pt rotary motor to people in the forum and asked if anyone could do anything with it. This model was my first attempt and many other people can up with even more creative ideas. Lonelyswedish built an especially nice walker with it.
082 name: Ontology
genre: walker/flex
comments: This model is a more complex version of Fundamental_Ontology. It uses flex circles for legs but the effect isn't quite what I desired. Ontology is the study of being and is the focus of my college career.
189 name: Optical_Fiber
genre: zero-g
comments: This model is very similar to Faberge_Egg and Shark_Teeth but uses much less wave amplitude than the latter. I thought it created a nice zero-g 3d effect, but it also functions well as a roller if you increase gravity.
061 name: OrangePentagon_KevinoStyle
genre: amoeba/flex
comments: A constructor named Roey_Schurr asked me to teach him what I knew about flex structures like Kevino's TriangleQ. Over the course of several emails I explained how to do it and I made this model to bridge the gap between my square and hexagon versions.
166 name: Orbit
genre: zero-g
comments: There is a very nice model in the zoo that is basically a big circle with a round curtain hanging off it of equal diameter. I forget the name but it creates a very satisfying effect. I tried to recreate this look using a couple of counter rotors.
037 name: Organelle
genre: zero-g/flex
comments: Organelle uses three of the same structure that Gemini_Twins used. The main difference is that they have some masses in common and so intertwine when they move.
131 name: Ottoman
genre: walker/motored
comments: This is a relatively simple example of how to power a walker with two identical rotary motors of any type. Basically you just construct reinforced squares to the corresponding masses and tune it. It's very easy.
273 name: Overkill
genre: walker/motored
comments: I'm not sure if people really understood this model. Basically it converts the linear motion from MultiMotors into rotary motion via the RAM linkages, then converts it back to linear motion by removing the vertical component with actuators. Then the legs are hooked up to regain elliptical motion.
322 name: Overwhelm
genre: walker/motored
comments: Overwhelm is related to Overkill in principle. It uses linear actuators to drive rotors but simultaneously cancels out one component of their elliptical motion. Then I used the resulting 3pt linear configurations to drive a conventional offset group of legs.
